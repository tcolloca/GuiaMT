Sistema de Puntuación

El sistema de puntuación fue el siguiente:

El 1º en resolver el puzzle, conseguía 4, 5, o 6 puntos si el enigma era de dificultad fácil, media o difícil respectivamente.
El 2º en resolver el puzzle, conseguía 3, 4, o 5 puntos si el enigma era de dificultad fácil, media o difícil respectivamente.
El 3º en resolver el puzzle, conseguía 2, 3, o 4 puntos si el enigma era de dificultad fácil, media o difícil respectivamente.
El resto al resolver el puzzle conseguía 1, 2, o 3 puntos si el enigma era de dificultad fácil, media o difícil respectivamente.
