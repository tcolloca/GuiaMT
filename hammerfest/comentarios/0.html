El primer nivel del juego es realmente fácil, pero para obtener el máximo beneficio hay tres "trucos", el primero es saltar en la plataforma superior izquierda, para obtener una moneda de oro (Ver Objetos). El segundo truco es que, al caer al pozo te inclines a la izquierda para acceder a un pasadizo secreto para tener beneficios en el siguiente nivel. Por ultimo puedes usar una bomba para derribar el cubo del pozo, esto hará que pases directamente al Nivel 10, esto parece muy bueno pero en realidad pierdes objetos valiosos, muchos puntos importantes, la posibilidad de ganar vidas y ante todo experiencia, la cuál es la clave para mejorar. Este nivel, también contiene la primera paralela del juego, la cuál requiere la Llave Maestra de Tubérculo.
