Introducción al HTML

Importante: Los links de la guía, deben ser puestos sin el http://www.guiamt.net, por lo tanto, si quieres redirigir a http:/www.guiamt.net/images/foto.png, simplemente pon /images/foto.png.

Básicamente, HTML consiste en etiquetas (<i>tags</i> en inglés), que son de la forma <nombre_etiqueta> para abrir, y </nombre_etiqueta> para cerrar, y entre ellas el contenido; pero también se admite <nombre_etiqueta/> para etiquetas vacías. Además, pueden tener atributos que las modifiquen, y será de la forma <etiqueta atributo1="valor1" atributo2="valor2" ... atributoN="valorN">. Se recomienda usar Adobe Dreamweaver para escribir HTML, o algún otro correcto/editor de HTML, pero es totalmente prescindible el uso de una herramienta tal. He aquí las etiquetas más importantes, sus efectos, y posibles atributos( Para ver todos los tags y saber más de ellos, visita esta página: w3schools):

  • <strong>negrita</strong>
  • <em>cursiva</em>
  • <u>subrayado</u> (No se recomienda usar, porque confunde con links)
  • <s>tachado</s>
  • <ins>insertado, luego de tachar</ins>
  • <p class="center">El parrafo aparecerá centrado</p>
  • <!--Comentario-->
  • Tabla: <table><!-- Fila1 --><tr><!-- Col1 --><td>Contenido</td><!-- Col2 --><td>Contenido</td></tr><!-- Fila2 --><tr><td>Cont</td><td>Cont</td></tr></table>. Para que una columna ocupe más columnas, se pone el atributo colspan="cant_de_cols_que_ocupa" . Para forzar altura y ancho de una tabla o fila, se pueden usar los atributos width="num" height="num". Para agregar distancia del contenido a las paredes de las celdas, se agrega en <table> el atributo cell-padding="cant". También se puede agregar el atributo align="posicion", donde la posicion puede ser right, left, o center.
  • <p>nuevo parrafo</p>
  • <br />salto de línea
  • <pre> </pre>gran espacio entre líneas
  • Viñeta: <ul><li>campo1</li><li>campo2</li>...<li>campN</li></ul>
  • <img src="/donde/esta/la/imagen.png" title="nombre_al_pasar_mouse_por_encima" alt="nombre_de_imagen" /> Nótese que los atributos son opcionales, pero se pide que se ponga el alt.
  • <a href="/a/la/cual/linkear">aqui va el texto que al cliquear te lleva a la pagina</a>
  • <subtitle>títulos</subtitle>*</li>
  • <minititle>subtítulos</minititle>*

*Nótese que estas etiquetas no existen realmente en HTML, pero fueron creadas en este sitio para que se usen ellas.
